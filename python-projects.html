Random Number Guessing Game

This project is based on a Python random number guessing game I created. The game generates a random number between 1 and 100 and gives the user hints until the correct number is guessed. After guessing, the number resets so the user can play again.

Halloween Hangman Game

This project is based on a Python Hangman game I created. The game randomly selects a Halloween-themed word (halloween, pumpkin, spooky, witch, ghost, haunted) and lets the user guess one letter at a time or guess the full word. It tracks incorrect guesses, prevents duplicate guesses, and displays a win or lose message based on the player’s guesses.
